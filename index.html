概述

什么是 Claude Code？

Claude Code 是 Anthropic 推出的终端 AI 编程代理工具，基于 Claude 大模型，以命令行方式深度参与你的开发工作流。

它直接运行在你的终端里，能读写文件、执行 Shell 命令、搜索代码库、管理 Git、运行测试，形成"感知→思考→行动→验证"的完整 Agent Loop。就像终端里多了一个随时待命的资深工程师，帮你排查 Bug、重构代码、编写测试、解释项目架构。

已被 Thoughtworks Technology Radar 评为 Adopt 级别——行业对 AI 编程助手的最高推荐水准。

快速开始

三步上手

从安装到开始对话，几分钟就能跑起来。

1

安装 Claude Code

推荐使用原生安装脚本，自动更新。

# macOS / Linux
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex

要求 Node.js 18+，Git 2.30+

2

首次认证

在项目目录下启动，浏览器会自动弹出 OAuth 登录页面。

cd my-project
claude

也可以设置 ANTHROPIC_API_KEY 环境变量用于无头环境

3

开始对话

看到 claude> 提示符后，用自然语言描述任务即可。

# 直接提问
"解释这个项目是做什么的"

# 带上下文操作
"给登录模块写单元测试"

# 管道模式
cat error.log | claude -p "分析这个报错"
命令速查

常用斜杠命令

在对话中输入这些命令，高效管理会话和项目。

命令用途
/help查看所有可用命令
/init初始化项目 CLAUDE.md 配置文件
/model切换模型（Sonnet / Opus / Haiku）
/plan进入计划模式——先出方案，不动代码
/compact压缩对话上下文，释放 Token 空间
/clear清除当前对话历史
/review代码审查当前更改
/cost查看本次会话费用和时长
/status查看会话状态和配置
/config打开配置面板
/vim切换到 Vim 编辑模式
/loop设置定期循环执行任务
效率技巧

常用快捷键

掌握这些快捷键可以大幅提升日常使用效率。

快捷键功能
Ctrl + C取消当前生成或操作
Ctrl + D退出 Claude Code 会话
Esc Esc撤销上一步文件修改
Shift + Tab切换权限模式（Plan / Auto / Manual）
Ctrl + L清屏
Up / Down浏览命令历史
Tab自动补全 / 接受建议
Ctrl + O切换到 Focus 视图（折叠详情）
