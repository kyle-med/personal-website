借助 Claude Code Agent 和 DeepSeek 模型的协作，从零开发了个人学术网站。项目采用 Astro 框架搭建、Tailwind CSS 渲染样式、Vercel 平台部署，实现了三个核心模块：

- **Academic Notes**：学术论文精读笔记，支持 Markdown 写作、标签筛选和全文搜索
- **Achievements**：个人进展时间线，按类型（论文/项目/演讲/奖项/里程碑）分类展示
- **Bedside Reference**：临床常用评分量表工具集，按科室分类，内置交互式计算器

从构思、设计、编码到上线全程通过 AI 辅助完成，包括架构设计、组件开发、搜索集成、GitHub 部署等完整链路。域名 hoikei-chan.vercel.app。
